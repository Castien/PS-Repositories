Form Validation

The required attribute defines fields that must be filled out before a form can be submitted.
The minlength and maxlength attributes define the length that an input must adhere to in order to be considered valid.
The min and max attributes define the value that an input must adhere to in order to be considered valid.
The pattern attribute provides powerful tools for matching an input to a desired format using regular expressions.
